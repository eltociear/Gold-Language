# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| < 1.1   | :x:                |

## Reporting a Vulnerability

We are happy to see that you want to Report a security issue!

To Report a Security Vulnerability, go to Issues, and create a new issue.
